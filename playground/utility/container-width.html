Container Width

The container width token sets the maximum width of the page content area. It responds to the current viewport: 360px (mobile) / 888px (tablet ≥ 1024px) / 1312px (desktop ≥ 1440px).

Container Width

SGDS Tailwind Token CSS Variable Mobile / Tablet / Desktop sgds:w-container --sgds-container-width 360px / 888px / 1312px

Example: Centred Page Container

Apply sgds:w-container sgds:mx-auto to a wrapper to keep page content within the responsive content width at every breakpoint.
