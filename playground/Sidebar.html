Sidebar Component Playground

Overview
This sidebar demonstrates a collapsible navigation component with support for:
  • Organized sections to group related options
  • Collapsible sections to save space
  • Up to 3 levels of nested options
  • Floating overlays for nested content
Components
  • sgds-sidebar - Main container component
  • sgds-sidebar-section - Organizes and groups options (NEW!)
  • sgds-sidebar-item - Individual navigation items
Features
  • Click the menu icon in the sidebar header to collapse/expand the entire sidebar
  • Click section headers to expand/collapse sections (if collapsible)
  • Hover over or focus on options with nested items to show floating overlay
  • Organize options with sidebar sections for better structure
  • Supports up to 3 levels of nesting within options
  • Selected state for active option (Dashboard)
  • Disabled state example (Premium Features)
  • Custom icons using sgds-icon component
  • Smooth animations and transitions
Sidebar Sections

The sidebar is organized into sections:

  • Main Section - Always expanded, contains Dashboard and Reports
  • Organization Section - Collapsible, contains Team Management and Projects
  • Configuration Section - Collapsible, contains Settings
  • Support Section - Always expanded, contains Help & Support and Premium Features
sgds-sidebar-section Properties
  • title (string) - The section header text displayed above the options
  • collapsed (boolean, default: false) - Whether the section is in collapsed state
  • collapsible (boolean, default: false) - If true, users can toggle the section open/closed

Example: <sgds-sidebar-section title="Organization" collapsible collapsed="true">...</sgds-sidebar-section>

Slot Features
  • Default slot - Label text for the option
  • before slot - Icon or content displayed before the label
  • after slot - Icon, badge, or indicator displayed after the label
  • nested slot - Nested options that appear in overlay on hover

The "Help & Support" option demonstrates the after slot with a badge indicator.

Interactive Examples

Try these interactions:

  • Hover over "Team Management" to see options with 3 levels of nesting
  • Hover over "Settings" to see nested Privacy & Security with 3rd level items
  • Hover over "Projects" to see multiple branches with nested items
  • Check the red badge on "Help & Support" - demonstrates the after slot
  • Click the collapse button to minimize the sidebar width
  • Try "Premium Features" - it's disabled and cannot be interacted with
