## Color Groupings

### Surface Colors

| SGDS Tailwind Token | CSS Variable | Purpose |
|---|---|---|
| `sgds:bg-surface-default` | `--sgds-surface-default` | Base surface color |
| `sgds:bg-surface-raised` | `--sgds-surface-raised` | Elevated surface (cards, panels) |
| `sgds:bg-surface-inverse` | `--sgds-surface-inverse` | Inverted surface with inverse text |
| `sgds:bg-surface-fixed-light` | `--sgds-surface-fixed-light` | Always light surface |
| `sgds:bg-surface-fixed-dark` | `--sgds-surface-fixed-dark` | Always dark surface |
