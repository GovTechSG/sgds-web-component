## Disabling validation on form level using native `novalidate` attribute

Alternatively, you can disable validation on the form level using the native `novalidate` attribute on the `<form>` element.

When `novalidate` is present on the closest parent `<form>`, all child form components will have their constraint validation and SGDS validation disabled.
