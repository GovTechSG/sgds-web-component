## Layout

Controls the CSS `table-layout` algorithm to determine how column widths are calculated.

### Auto

Use `layout="auto"` (default) to let the browser size columns based on content. Columns expand or contract to fit their content naturally.

<Canvas of={TableStories.LayoutAuto}>
  <Story of={TableStories.LayoutAuto} />
</Canvas>

### Fixed

Use `layout="fixed"` to distribute column widths evenly regardless of content, providing uniform column sizing.

<Canvas of={TableStories.LayoutFixed}>
  <Story of={TableStories.LayoutFixed} />
</Canvas>

### Fixed with custom column widths

When using `layout="fixed"`, you can assign arbitrary widths to individual `<sgds-table-head>` cells using the `class` attribute with SGDS utility classes (e.g. `class="sgds:w-50"`). Columns without an explicit width will share the remaining space equally.
