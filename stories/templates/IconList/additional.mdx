## Sizes

The default icon list size is medium. Add `size="lg"`,` size="sm"` for additional sizes.

- `size="sm"`: Reduces the icon and text size.
- Default (`size="md"`): Standard icon and text size, applied when no `size` attribute is specified
- `size="lg"`: Increases both the icon and text size.
