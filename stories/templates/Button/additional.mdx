## Variants

Use any of the available button style types to quickly create a styled button. Modify the `variant` prop to the intended value.

<Canvas>
  <Story name="Variants">
    {["primary", "secondary", "success", "danger", "warning", "info", "light", "dark"].map(
      variant => html` <sgds-button .variant=${variant}> ${variant} </sgds-button> `
    )}
  </Story>
</Canvas>

## Outline buttons

For a lighter touch, Buttons also come in `outline-*` variants with no background color. Modify the `variant` prop to the intended value.

<Canvas>
  <Story name="Outline buttons">
    {[
      "outline-primary",
      "outline-secondary",
      "outline-success",
      "outline-danger",
      "outline-warning",
      "outline-info",
      "outline-light",
      "outline-dark"
    ].map(variant => html` <sgds-button .variant=${variant}> ${variant} </sgds-button> `)}
  </Story>
</Canvas>

## Sizes

Fancy larger or smaller buttons? Add size="lg", size="sm" for additional sizes.

<Canvas>
  <Story name="Sizes">
    {["sm", "lg"].map(size => html` <sgds-button .size=${size}> ${size}-button </sgds-button> `)}
  </Story>
</Canvas>

## Active state

To set a button's active state simply set the component's `active` prop.

<Canvas>
  <Story name="Active state">
    {["primary", "secondary"].map(
      variant => html` <sgds-button .variant=${variant} active> ${variant}-button </sgds-button> `
    )}
  </Story>
</Canvas>
