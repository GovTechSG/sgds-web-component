## File States

Control the upload state of individual files programmatically.

```typescript
public setFileUploadState(
  index: number,
  state: 'loading' | 'success' | 'error',
  error?: string
): void
```

### Parameters

- **index** - The file index in the list (0-based)
- **state** - The upload state: `'loading'`, `'success'`, or `'error'`
- **error** - Error message to display (only used with `'error'` state)

## Custom Validation with noValidate

Disable SGDS's built-in validation to implement custom validation logic using your own validation library (Zod, Yup, VeeValidate, etc).

<Canvas of={FileUploadStories.CustomValidation} />

Set the `noValidate` prop to opt out of native HTML5 and SGDS validation:

```html
<sgds-file-upload
  noValidate
  required
  hasFeedback
  multiple
  accept=".pdf"
  label="Upload Documents"
  name="documents"
  hintText="Max 2 PDF files. SGDS validation is disabled, using custom validation instead"
>
  Choose files
</sgds-file-upload>
```

Then handle validation manually via events:

```js
const fileUpload = document.querySelector('sgds-file-upload');

fileUpload.addEventListener('sgds-add-files', (e) => {
  const files = e.detail;

  // Custom validation rules:
  // 1. Max 2 files total
  // 2. PDF only

  const allFiles = fileUpload.files;
  let isValid = true;
  let errorMsg = "";

  if (allFiles.length > 2) {
    isValid = false;
    errorMsg = "Maximum 2 files allowed";
  }

  for (const file of files) {
    if (!file.name.toLowerCase().endsWith(".pdf")) {
      isValid = false;
      errorMsg = "Only PDF files are allowed";
      break;
    }
  }

  fileUpload.invalidFeedback = errorMsg;
  fileUpload.setInvalid(!isValid);
});

fileUpload.addEventListener('sgds-remove-file', (e) => {
  // Re-validate remaining files
  const remaining = e.detail.files;
  if (remaining.length === 0) {
    fileUpload.invalidFeedback = "At least one file is required";
    fileUpload.setInvalid(true);
  } else {
    fileUpload.setInvalid(false);
  }
});
```

### Alternative: Form-level novalidate

Disable validation for all form components at once:

```html
<form novalidate>
  <sgds-file-upload required name="doc"></sgds-file-upload>
  <sgds-input required name="name"></sgds-input>
  <!-- All SGDS components auto-detect form novalidate -->
</form>
```

> **Note:** When `noValidate` is true, native HTML5 validation is disabled. Only your custom validation runs. Use the `sgds-add-files` and `sgds-remove-file` events to implement your validation rules.
