## File States

Control the upload state of individual files programmatically.

```typescript
public setFileUploadState(
  index: number,
  state: 'uploading' | 'success' | 'error',
  error?: string
): void
```

### Parameters

- **index** - The file index in the list (0-based)
- **state** - The upload state: `'uploading'`, `'success'`, or `'error'`
- **error** - Error message to display (only used with `'error'` state)

## Events

### sgds-add-files

Fires when files are added to the upload. Use this to initiate uploads.

<Canvas of={FileUploadStories.SgdsAddFiles} />

### sgds-remove-file

Fires when a file is deleted. Includes the deleted file and remaining files.

<Canvas of={FileUploadStories.SgdsRemoveFile} />

### sgds-change

Fires on any file set change (add or remove). Useful for tracking current state.

> **Note:** Use `sgds-change` instead of the deprecated `sgds-files-selected` event for tracking all file set changes.

<Canvas of={FileUploadStories.SgdsChange} />
