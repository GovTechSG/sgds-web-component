# Switch

Switch component is used to toggle on and off or yes or no action.

<Canvas>
  <Story of={SwitchStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsSwitch  from "@govtechsg/sgds-web-component/react/switch/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Switch";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Switch/index.umd.min.js"></script>
```

## Label 

Add label to provide textual hints on what the switch does. By default label is added to the right of the switch in the default slot.

Use `slot="leftLabel"` to add label to the left of the switch
