# Icon

Icons offer a form of visual shorthand that we are all familiar with. They can label, inform and aid navigation quickly and effectively in minimal space. Icons must first and foremost communicate meaning. By default, the icon component renders icons from `SgdsIcon` library set

<Canvas>
  <Story of={IconStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsIcon  from "@govtechsg/sgds-web-component/react/icon/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Icon";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Icon/index.umd.min.js"></script>
```

## Sizes

SgdsIcons offer a variety of sizes : `sm` , `md` , `lg` , `xl` , `2-xl` , `3-xl`. By default, the icon size is `lg`
