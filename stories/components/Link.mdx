# Link

Link allows users to click and navigate their way from page to page

<Canvas>
  <Story of={LinkStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsLink  from "@govtechsg/sgds-web-component/react/link/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Link";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Link/index.umd.min.js"></script>
```
