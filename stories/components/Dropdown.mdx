# Dropdown

`SgdsDropdownItem` are navigation links built with `HTMLAnchorElement`. It should be used in the default slot of `SgdsDropdown`<br/>`SgdsDropdown` toggles contextual overlays for displaying lists of links.

## Import

### React

    ``` jsx
    import SgdsDropdownItem  from "@govtechsg/sgds-web-component/react/dropdown-item/index.js";
    ```

    ``` jsx
    import SgdsDropdown  from "@govtechsg/sgds-web-component/react/dropdown/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Dropdown";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Dropdown/index.umd.min.js"></script>
```

## sgds-select event

The `sgds-select` event is emitted on `sgds-dropdown` whenever a non-disabled item is clicked. Use `event.detail.item` to access the selected `SgdsDropdownItem` element.

Disabled items do not emit `sgds-select`.
