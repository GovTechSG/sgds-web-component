# Checkbox

CheckboxGroup is a form component for multiselection of checkboxes.<br/>Checkbox component is used when you require users to select multiple items from a list.

## Import

### React

## Invalid and feedback

Each `sgds-checkbox` component controls its own `invalid` state.
To enable SGDS form validation styles set `hasFeedback` to true.

## Validation

### CheckboxGroup 

HTML form constraint validation is built-in. By default, there is no need to control the `invalid` prop to enable constraint validations. 

- To enable invalid styles and feedback message, set `hasFeedback` to true on `sgds-checkbox-group`.

In this example, press submit to show constraint validation behaviour for `required` on the checkboxes. 
Form reset reset's the validity and the value of the input to its initial default value.

  CheckboxGroup constraint validation works as a single form entity. 
  When `required` is set to true on CheckboxGroup, user has to minimally check one option for a valid submission. 
  Do not set `required` on the Checkbox children when using CheckboxGroup.
