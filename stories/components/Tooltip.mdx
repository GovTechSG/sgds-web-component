# Tooltip

Tooltips display more information when users hover over, focus on, or interact with an element.

<Canvas>
  <Story of={TooltipStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsTooltip  from "@govtechsg/sgds-web-component/react/tooltip/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Tooltip";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Tooltip/index.umd.min.js"></script>
```
