# Select

SelectOption is the option of the Select<br/>Select is used to make one selection from a list through keyboard or mouse actions

<Canvas>
  <Story of={SelectStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsSelectOption  from "@govtechsg/sgds-web-component/react/select-option/index.js";
    ```

    ``` jsx
    import SgdsSelect  from "@govtechsg/sgds-web-component/react/select/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Select";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Select/index.umd.min.js"></script>
```
