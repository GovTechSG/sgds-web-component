# CloseButton

Close button for closing actions. Used in Modal, Drawer, Alert and Toast.

<Canvas>
  <Story of={CloseButtonStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsCloseButton  from "@govtechsg/sgds-web-component/react/close-button/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/CloseButton";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/CloseButton/index.umd.min.js"></script>
```

# Additional Usage for sgds-close-button

This section demonstrates the `size` and `tone` props for the `sgds-close-button` component.

## Sizes

Use the `size` prop to render different button sizes:
