# Divider

A divider is a thin line that groups content in lists and layouts. They bring clarity to a layout by grouping and dividing content in close proximity.

## Import

### React

    ``` jsx
    import SgdsDivider  from "@govtechsg/sgds-web-component/react/divider/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Divider";
```

## Vertical orientation 

By default, the divider is of horizontal orientation.

Change to vertical orientation by setting `orientation="vertical"`.
Ensure that the parent container has a height.
