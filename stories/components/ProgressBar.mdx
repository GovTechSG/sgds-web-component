# ProgressBar

Provide up-to-date feedback on the progress of a workflow or action with simple yet flexible progress bars.

<Canvas>
  <Story of={ProgressBarStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsProgressBar  from "@govtechsg/sgds-web-component/react/progress-bar/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/ProgressBar";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/ProgressBar/index.umd.min.js"></script>
```
