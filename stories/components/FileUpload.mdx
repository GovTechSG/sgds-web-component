## Import

### React

    ``` jsx
    import SgdsFileUpload  from "@govtechsg/sgds-web-component/react/file-upload/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/FileUpload";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/FileUpload/index.umd.min.js"></script>
```
