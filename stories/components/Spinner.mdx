# Spinner

Spinners notify the users that their request is being processed.

<Canvas>
  <Story of={SpinnerStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsSpinner  from "@govtechsg/sgds-web-component/react/spinner/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Spinner";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Spinner/index.umd.min.js"></script>
```
