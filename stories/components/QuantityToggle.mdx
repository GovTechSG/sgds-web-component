# QuantityToggle

The quantity toggle component is used to increase or decrease an incremental venue, best used when the user needs to enter or adjust the quantity of a selected item.

  <Story of={QuantityToggleStories.Basic} />

## Import

### React

    ``` jsx
    import SgdsQuantityToggle  from "@govtechsg/sgds-web-component/react/quantity-toggle/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/QuantityToggle";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/QuantityToggle/index.umd.min.js"></script>
```

## Validation 

HTML form constraint validation is built-in. By default, there is no need to control the `invalid` prop to enable constraint validations. 

- To enable invalid styles and feedback message set `hasFeedback="both"`
- To enable invalid styles only, set `hasFeedback="style"`
- To enable feedback message only, set `hasFeedback="text"`

In this example, `hasFeedback` is set to `both`. 

By default, QuantityInput always have a default value of 0, `required` constraint validation is not applicable. There is no `required` prop. 
Instead, apply `min` and `max` constraint validation when needed. 

Resetting the form, reset's the validity and the value of the input to its initial default value.
