# Toast

ToastContainer is the container component to position `sgds-toast` in screen. When there is multiple toasts in the container, the toast components are stacked vertically.<br/>Toast allows you to convey quick messaging notifications to the user.

<Canvas>
  <Story of={ToastStories.Basic} />
</Canvas>

## Import

### React

    ``` jsx
    import SgdsToastContainer  from "@govtechsg/sgds-web-component/react/toast-container/index.js";
    ```

    ``` jsx
    import SgdsToast  from "@govtechsg/sgds-web-component/react/toast/index.js";
    ```

### Others (Vue, Angular, plain HTML etc.)

```js
import "@govtechsg/sgds-web-component/components/Toast";
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.16.0/components/Toast/index.umd.min.js"></script>
```

## Position with toast containers

ToastContainers positions the toast with respect to the entire page. Adjust the `position` prop accordingly to render your toast at the correct placements
Top position takes into consideration the height of masthead and mainnav.
